# `@sceneforge/scene` Package

The `@sceneforge/scene` package provides some utilities to handle 3D scenes
wrapping the Babylon.js library. The package is organized into the following:

- [Engine](?path=/docs/sceneforge-scene-engine--documentation)
- [Handler](?path=/docs/sceneforge-scene-handler--documentation)
- [Object](?path=/docs/sceneforge-scene-object--documentation)
- [Types](?path=/docs/sceneforge-scene-types--documentation)
